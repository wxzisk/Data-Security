## 作业题目

### 正确性验证（调用标准库验证）

如上文，由于环境问题，在这一小节中，没有调用标准库来进行正确性验证。但是我们通过将明文放到在线的SM4加密工具上进行了验证，并且也比对了老师提供的文件的结果。结果仍然是正确的。
